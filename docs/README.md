# Mercuri Website (Componentized)

## Run locally

From this directory:

```bash
python3 -m http.server 8080
```

Open `http://localhost:8080`.

## Structure

- `index.html`: page shell
- `src/main.js`: component registry
- `src/components/*.js`: section components
- `src/tailwind-config.js`: design tokens
- `src/styles.css`: shared styles
